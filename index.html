1. Scope of Information Collection

All data generated within the Game is only temporarily stored locally on your device. The Game will never upload any locally generated user data to our independent servers. Only TikTok’s official advertising SDK may collect standard device identifiers for ad delivery purposes. Such data is governed exclusively by TikTok’s separate privacy policy and is outside our independent control.

We only collect the following limited categories of local data necessary to support normal Game operation:

  • Local game operational data: game progress, merge records, in-game preference settings and local language configurations;
  • Temporary basic device information (read locally only, no external upload): device system language, device model, operating system version and local access timestamps;
  • Standard advertising identifiers requested by TikTok’s ad SDK for ad matching. Yego Digital LLC cannot independently access, store, extract or retain complete raw advertising identifiers.

The Game does NOT request or access any sensitive personal data, including health records, contact lists, photo galleries, precise geolocation, payment credentials, biometric data or other confidential personal information.

Important Statement on Local Data Retention

All game progress and preference records are saved solely on your device’s local storage. We do not provide any cloud backup services:

  • If you clear TikTok cache data, uninstall TikTok, or erase local storage data for this Game, all game progress and local records will be permanently deleted and cannot be recovered.
  • Game save files cannot be synced across different devices after you reinstall TikTok or switch devices.

No collected data will be used to build independent user profiles, send unsolicited marketing communications, or conduct cross-device tracking for commercial purposes. The Game does not collect any payment or billing-related data.
